# 嘉立创 EDA 工程文件

本目录保存温湿度闹钟复刻项目的嘉立创 EDA 原始工程包，文件可以直接使用嘉立创 EDA 打开。

| 文件 | 说明 |
| --- | --- |
| `temperature-humidity-alarm-clock.epro2` | 完整工程包，包含项目工程数据和配套资源。 |
| `temperature-humidity-alarm-clock-schematic.epro2` | 独立原理图工程包。 |
| `temperature-humidity-alarm-clock-pcb.epro2` | 独立 PCB 工程包。 |

`.epro2` 是嘉立创 EDA 的工程压缩包格式。下载后保留文件扩展名，使用嘉立创 EDA 的“打开工程”功能载入。
